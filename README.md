
# Weather App

A full-stack weather application that fetches real-time data from OpenWeatherMap API. Built with Node.js backend and vanilla HTML/CSS/JS frontend.

## Features

- Real-time weather data (temperature, humidity, wind speed, conditions)
- Search by city name
- Loading states and error handling (invalid city, API errors)
- Responsive design (mobile + desktop)
- Full-stack: custom Node.js backend + frontend

## Tech Stack

| Layer | Technology |
|-------|------------|
| Backend | Node.js, Express |
| Frontend | HTML, CSS, Vanilla JavaScript |
| API | OpenWeatherMap |
| Deployment | Github |

## Quick Start

```bash
git clone https://github.com/GreatKingdavid/Weather-App
cd Weather-App
npm install
node app.js

Then open http://localhost:3000 in your browser.

Environment Variables

Create a .env file:
API_KEY=your_openweathermap_api_key
PORT=3000
